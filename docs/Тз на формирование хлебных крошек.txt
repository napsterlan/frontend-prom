Тз на формирование хлебных крошек

1. Хлебные крошки формируются на основе пути в url
2. Названия крошек берутся из данных полученных через api
3. Выводим все крошки в виде ссылок с '/' между ними 
4. Последняя крошка не должна быть ссылкой, она должна быть названием категории или продукта
5. Один продукт может быть в нескольких категориях, поэтому необходимо запоминать откуда пришел пользователь (последняя категория) и выводить крошки по этому пути.
6. Если пользователь напрямую зашел на продукт (из поиска, например), то крошки формируются от последней категории, в которой лежит продукт
7. Для разделов проекты, новости, информационные страницы крошки формируются от корня

Примеры крошек:
1. /catalog/category1 -> [Главная, Каталог, category1]
2. /catalog/category1/category2 -> [Главная, Каталог, category1, category2]
3. /catalog/category1/product1 -> [Главная, Каталог, category1, product1]
4. /catalog/category1/category2/product2 -> [Главная, Каталог, category1, category2, product2]
5. /projects/categoryProjects/project1 -> [Главная, Проекты, categoryProjects, project1]
6. /news/news1 -> [Главная, Новости, news1]
7. /about -> [Главная, О компании]
8. /contacts -> [Главная, Контакты]
