Промпт
Действуй как backend разработчик.
Напиши основу backend проекта на golang. Используй gin, gorm, jwt. 
База данных postgres.
Описание структуры базы данных в файле database-struct.txt
Описание эндпоинтов в файле endpoints.txt
Описание ответов в файле response.txt

Начни с создания миграций для базы данных.
Для каждого эндпоинта создай отдельный файл в папке handlers.
В файле main.go создай роуты для эндпоинтов.
В файле models создай структуры данных по структурам из database-struct.txt
