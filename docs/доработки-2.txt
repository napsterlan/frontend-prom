Список задач по фронтенду:

### 1. Разработка базовой структуры приложения
+ Создание базовой структуры Next.js приложения с TypeScript
- Настройка Tailwind CSS
+ Настройка React Query для работы с API
+ Создание базовых компонентов (Layout, Header, Footer)

### 2. Система маршрутизации
+ Реализация динамической маршрутизации для категорий и продуктов
/catalog - главная страница каталога
/catalog/:slug/:slug - иерархический маршрут по вложенным категориям

/product/:slug - маршрут к продукту, 
где slug - продукт

+ Разработка системы хлебных крошек с учетом вложенности категорий
- Реализация обработки прямых переходов на продукты

### 3. Разработка компонентов каталога
+ Создание компонента списка категорий с поддержкой вложенности
+ Создание компонента табличного списка продуктов с возможностью выбора характеристик в качестве колонок
- Реализация фильтрации и сортировки продуктов. (Прием параметров через URL)
- Создание компонента пагинации
- Разработка компонента карточки продукта

### 4. Интеграция с API
- Создание сервисного слоя для работы с API
- Реализация кэширования запросов с помощью React Query
- Разработка обработчиков ошибок API
- Типизация ответов API согласно документации:

- Внедрение серверного рендеринга для SEO
- Настройка кэширования статических данных

### 6. Компоненты управления данными
- Разработка форм создания/редактирования новостей
- Создание форм управления проектами
- Реализация загрузки и управления изображениями
- Разработка компонентов для работы с метаданными

### 7. Система авторизации
