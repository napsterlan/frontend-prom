/catalog - главная страница каталога
/catalog/:slug/:slug - иерархический маршрут по вложенным категориям

/catalog/:slug/:slug - маршрут к продукту, 
где первый slug - категория (категорий может быть несколько), второй slug - продукт

/news - главная страница новостей
/news/:slug - страница одной новости

/projects - главная страница проектов
/projects/:slug - страница категории проектов
/projects/:slug/:slug - страница одного проекта

/info - главная страница информационных страниц
/info/:slug - страница одной информационной страницы
